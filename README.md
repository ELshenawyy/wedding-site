# Mostafa & Sara — Wedding Site

A single static page (no build step). Open `index.html` directly in a browser to preview.

## To customize
- Replace the hero background: drop a photo at `images/hero.jpg` and update `.hero-bg` in `style.css` to use `background-image: url('images/hero.jpg')`.
- Fill in the `<!-- TODO -->` placeholders in `index.html` (love story text, ceremony time).

## Deploy on GitHub Pages
1. Push this folder to a new GitHub repo.
2. Repo Settings → Pages → Deploy from branch → `main` / root.
3. Your site will be live at `https://<username>.github.io/<repo>/`.
